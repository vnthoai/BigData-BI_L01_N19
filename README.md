# BigData-BI_L01_N19
(HCMUT) BigData&amp;BI_L01 Group19 Assignment

Dataset: Social Media Engagement
Nguồn: Kaggle
https://www.kaggle.com/datasets/subashmaster0411/social-media-engagement-dataset

Bước 1: Cài đặt và Thiết lập Kaggle API

Tạo API Key: Truy cập trang tài khoản Kaggle của bạn, cuộn xuống phần "API", và nhấn "Create New API Token". Tệp kaggle.json sẽ được tải về.

Cài đặt Kaggle: !pip install kaggle

Thiết lập API Key: Tải tệp kaggle.json lên môi trường của bạn (hoặc đặt nó vào thư mục .kaggle trên máy tính cục bộ).

Bước 2: Tải Dataset

Sử dụng câu lệnh sau để tải dataset trực tiếp: !kaggle datasets download -d subashmaster0411/social-media-engagement-dataset
